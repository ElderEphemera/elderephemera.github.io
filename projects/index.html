Projects

Below is a list of my projects that are hosted on this site. For other projects, check out my GitHub page.

  • An Ark for the Amigalites

    A small fishing game where you can listen to music and create image collages. This was created in a week for the DiscMaster game jam. 100% of the assets come from DiscMaster, a search engine for vintage data hosted on the Internet Archive.

  • Four

    Something approaching a 2048-esque puzzle game. But it doesn't quite achieve the "puzzle" or even the "game" part. Originally created for the Not A Game Jam Game Jam.

  • Recipe Bookmarks

    A minimalistic website for cleanly displaying recipes from anywhere on the internet. Designed to be used with browser bookmarks so you can keep track of your favorite recipes.
